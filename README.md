По-смотри в окно
Проект «По-смотри в окно» — интерфейс для просмотра видео с фильтрацией по городу и времени суток.
Сделан в духе Яндекс-проектов с использованием HTML, CSS и JavaScript.

Структура проекта
/project-root

│

├─ index.html # Главная страница

├─ styles/

│ └─ style.css # Основной CSS

├─ scripts/

│ └─ script.js # Логика формы и работы с видео

├─ assets/

│ ├─ images/ # Картинки/постеры видео

│ └─ videos/ # Видео для теста

└─ README.md

Технологии
HTML5
CSS3 (Flexbox, кастомные чекбоксы, псевдоэлементы)
JavaScript (Vanilla JS, работа с шаблонами <template>, DOM)
Шрифты: 'Fira Sans Condensed' и 'Oswald'
Установка и запуск
Клонируйте репозиторий:
git clone https://github.com/K4rens/posmotri-v-okno-ad
Откройте index.html в браузере (Chrome, Firefox или Edge).
Использование
Введите город в поле поиска.

Выберите время суток с помощью чекбоксов.

Нажмите «Найти» для фильтрации видео.

Просмотр доступен прямо на странице без перезагрузки.

Замечания по коду
Все элементы формы и карточки видео создаются динамически с использованием в JS.

Кастомные чекбоксы реализованы через span.search-form\_\_pseudo-checkbox.

Форма и видео выровнены по сетке с помощью .content и flexbox.
